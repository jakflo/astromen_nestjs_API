Backend aplikace v Nest.js

## Instalace
- nainstalujte závislosti pomocí npm install
- spusťte DB a importujte z astromen_nestjs.sql
- je-li potřeba, nastavte připojení k DB v config.js
- spusťe dev server pomocí npm run start:dev
- nebo zkompilujte produkci pomocí npm run build a spusťte produkčí server pomocí npm run start:prod
